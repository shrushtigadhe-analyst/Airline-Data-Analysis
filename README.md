# Airline-Data-Analysis
Exploratory Data Analysis of flight dataset using Python (Pandas, Matplotlib, Seaborn).
Airline Data Analysis 

This project explores flight dataset using Python (Pandas, Matplotlib, Seaborn).

 Key Questions
- Which airline operates the most flights?
- Which airline carries the most passengers?
- Top 10 airlines by passengers
- Top 10 airlines by revenue

Visualizations
- Flights per airline (bar & pie chart)
- Revenue by airline (bar chart)
- Delay distribution (histogram)
- Flights over time (line chart)

Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

 Insights
- The dataset reveals flight frequency patterns and delay trends.
- Airlines with higher passenger counts often correlate with higher revenue.
- Delay distribution shows most flights arrive within 0–100 minutes delay.

 How to Run
1. Download the notebook (`flights.ipynb`)
2. Open it in Jupyter Notebook or VS Code
3. Run all cells sequentially to view analysis and charts
